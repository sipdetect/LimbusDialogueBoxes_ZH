# LimbusDialogueBoxes_ZH

## 罪人語音氣泡華語版

本專案為《Limbus Company》戰鬥語音氣泡華語翻譯版本。  
戰鬥氣泡原始檔案由 **NotherWael** 提供，翻譯規則與文本由 **零協會** 整理與維護。

歡迎協助糾錯與補正，共同打造更完整的戰鬥閱讀體驗。

---

### 4️⃣ 進入翻譯文本資料夾

請進入你目前使用的翻譯資料夾。  
（此範例為個人版本，零協會或其他提供者可能使用不同資料夾名稱。）

<img width="758" height="198" alt="圖片" src="https://github.com/user-attachments/assets/60d34614-0a2d-44c1-be1a-c4fb9ff3f218" />

---

### 5️⃣ 放入語音氣泡檔案

將本專案提供的以下兩個檔案放入該資料夾：

```
BattleSpeechBubbleDlg.json
BattleSpeechBubbleDlg_mowe.json
```

<img width="991" height="565" alt="圖片" src="https://github.com/user-attachments/assets/f025c8ed-e7d9-4027-a51e-937048533c90" />

完成後重新啟動遊戲即可生效。
